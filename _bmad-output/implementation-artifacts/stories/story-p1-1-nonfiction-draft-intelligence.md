# Story P1.1: Deepen Nonfiction Draft Intelligence

Status: draft

## Story

As an author,
I want nonfiction chapter drafts to feel woven and authored,
so that the output reads like a coherent book rather than a staged assembly.

## Acceptance Criteria

1. Chapter drafts synthesize Promise, Outline, Base Story, Research, External Stories, and Personal Stories more naturally.
2. The system detects weak argument support, repetition, and thin evidence.
3. Underused upstream material is surfaced before draft commit.
4. Chapter-level quality scoring is available.
